# StrokeSense Backend

StrokeSense is a capstone project web application for early stroke risk screening.

This repository contains the backend API built with Node.js and Express.js.

## Backend Owner

Luthfi Wiradhika Ananta

## Project Status

The backend currently supports:

- Running locally
- Health check endpoint
- Form field options endpoint
- Prediction endpoint
- Backend input validation
- Mock stroke risk prediction
- Temporary prediction history storage
- API contract documentation for frontend integration

The AI model is not connected yet because the AI Engineer team is still preparing their model/API.

## Tech Stack

- Node.js
- Express.js
- Zod
- CORS
- Helmet
- Morgan
- PostgreSQL-ready storage
- RESTful API
